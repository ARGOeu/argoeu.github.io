Overview

ARGO is a lightweight service for Service Level Monitoring designed for medium and large sized Research Infrastructures.

The ARGO software stack is comprised of the following components (products)

  • Sync components
  • Poem service
  • Consumer
  • Compute Engine
  • API
  • Web UI
